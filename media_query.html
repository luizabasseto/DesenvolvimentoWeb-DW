Media Query

  • O que são medias queries?

    •       Uma media query consiste em um tipo de mídia e pelo menos uma expressão que usa características de mídia como largura, altura e cor para limitar o escopo de uma folha de estilo. As consultas de mídia adicionadas no CSS3 adaptam a renderização do conteúdo a uma variedade de dispositivos sem alterar o conteúdo em si.

          Uma consulta de mídia consiste em um tipo de mídia e pode conter uma ou mais expressões de acordo com a especificação CSS3, expressa em termos de características da mídia, que determinam verdadeiro ou falso. O resultado da consulta será verdadeiro se o tipo de mídia especificado na consulta de mídia corresponder ao tipo de documento exibido no dispositivo e todas as expressões na consulta de mídia forem verdadeiras. Quando a consulta de mídia for verdadeira, a camada de estilo ou as regras de estilo correspondentes serão aplicadas, seguindo o padrão de regras em cascata. Camadas de estilo com consultas de mídia vinculadas a tags <link> serão baixadas mesmo se a consulta de mídia retornar falsa. A menos que você use os operadores not ou only, o media type é opcional e o tipo all será implícito.

    <!-- CSS media query em um elemento de link -->
    <link rel="stylesheet" media="(max-width: 800px)" href="example.css"/>
    <!-- CSS media query dentro de um stylesheet -->
    @media (max-width: 600px){
      .facet_sidebar{
        display: none;
      }
    }
